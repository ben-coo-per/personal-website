The app is built as a PWA, allowing users to access it from either their web browser or by downloading it on their phone like a mobile app. The new rsvp system has increased the number of respondents to our weekly meetings, which allows us to make decisions about meeting times.

![Gallery image 1](/cms-assets/projects/reading-group-app/gallery-1.png)

We have been using the app since July 2023, throughout this time I toyed with a few new features such as chat - however, I learned that there is too much friction to moving off of our existing chat app (Groupme) and have pivoted to building a Groupme integration.
