# Example Blog Post

This is an example blog post demonstrating the new Kirby-based blog system.

## Introduction

The blog has been migrated from Sanity CMS to a flat-file structure using Kirby's text format for metadata and Markdown for content.

## Features

- **Simple file structure**: Each post is a folder with `blog.txt` for metadata and `content.md` for the body
- **Local assets**: Images and files can be placed in the post folder
- **Markdown support**: Full Markdown rendering with custom components

## Conclusion

This new system is simpler to maintain and doesn't require external services.
